# Моделювання кидків кубиків методом Монте-Карло


## Аналітичні розрахунки

При киданні двох кубиків можливі суми від 2 до 12. Загальна кількість комбінацій: 6 × 6 = 36.

| Сума | Ймовірність   |
|:----:|:-------------:|
| 2    | 2.78% (1/36)  |
| 3    | 5.56% (2/36)  |
| 4    | 8.33% (3/36)  |
| 5    | 11.11% (4/36) |
| 6    | 13.89% (5/36) |
| 7    | 16.67% (6/36) |
| 8    | 13.89% (5/36) |
| 9    | 11.11% (4/36) |
| 10   | 8.33% (3/36)  |
| 11   | 5.56% (2/36)  |
| 12   | 2.78% (1/36)  |

## Результати методу Монте-Карло

Експерименти проводились з різною кількістю кидків: 100, 500, 1000, 5000, 10000.

Формат виводу програми: `X.XX% (Y/36)`, де Y — округлене значення ймовірності, помножене на 36.

## Висновки

### Підтверджено правильності розрахунків

**Метод Монте-Карло підтверджує аналітичні розрахунки.** При збільшенні кількості симуляцій результати все більше наближаються до теоретичних значень.

### Залежність точності від кількості експериментів

| Кількість кидків | Середня похибка | Характеристика точності |
|:----------------:|:---------------:|:-----------------------:|
| 100              | ~2-3%           | Значні відхилення       |
| 500              | ~1-1.5%         | Помітні відхилення      |
| 1000             | ~0.5-1%         | Прийнятна точність      |
| 5000             | ~0.2-0.5%       | Хороша точність         |
| 10000            | ~0.1-0.3%       | Висока точність         |
